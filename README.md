# 🤔到底有多大？

## 前言

一个用于把 PCB 在屏幕上的长度精确映射为现实尺寸的小工具。通过在屏幕上标定一段已知长度的线段，计算并应用缩放系数，从而可以以真实尺寸预览。只需对一个显示器测量一次即可保存校准数据，支持暗色模式，界面简洁、轻量并且离线运行。

![](./images/day.png)

## 用法

1. 安装插件，勾选配置中的“显示在顶部菜单”，并打开插件

![](./images/setup.png)

2. 点击步骤一的按钮，在PCB上放置一个线段，如果在显示器上过长或过短，可以对参数进行相应的调整

3. 量出线段在你的显示器上的实际长度

![](./images/measure.jpg)

4. 将量出的实际长度填入输入框

5. 点击执行缩放

6. Enjoy

![](./images/show.jpg)
